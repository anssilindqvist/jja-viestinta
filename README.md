# JJA Viestintä

Järvenpään Jousiampujien epävirallisen Instagram-tilin [@jja.1975](https://www.instagram.com/jja.1975/) sisällöntuotannon ylläpito ja suunnittelu.

> Tämä repo on viestintätiimin yhteinen työtila. Täältä löytyvät julkaisuaikataulu, Reels-käsikirjoitukset ja Q&A-sisällöt. Sisältösuunnitelma ja brändiohjeet ovat erillisessä claude chatissä.

---

## 📁 Mitä täältä löytyy

| Tiedosto | Mitä sisältää |
|----------|---------------|
| **[SCHEDULE.md](SCHEDULE.md)** | Päivittäinen julkaisuaikataulu — mitä on julkaistu, mitä tulee seuraavaksi, sisältöideat |
| **[REELS.md](REELS.md)** | Reels-videoiden käsikirjoitukset ja tuotantostatukset |
| **[QA.md](QA.md)** | Q&A-kysymykset ja vastaukset — Storyihin ja Highlighteihin tuleva sisältö |
| **[CONTRIBUTING.md](CONTRIBUTING.md)** | Ohjeet tiimille: miten päivitetään tiedostoja, mitä statuksia käytetään, julkaisurutiini |

---

## 🚀 Nopea aloitus tiimiläiselle

1. **Kirjaudu GitHubiin** — sinut on kutsuttu tähän repoon sähköpostitse
2. **Lue [CONTRIBUTING.md](CONTRIBUTING.md)** — siitä löytyvät ohjeet päivittämiseen
3. **Klikkaa mitä tahansa .md-tiedostoa** nähdäksesi sisällön kauniisti muotoiltuna
4. **Muokkaa tarvittaessa** — klikkaa kynä-ikonia (✏️) oikeasta yläkulmasta

Et tarvitse git-taitoja. Kaikki onnistuu selaimessa.

---

## 📅 Tämän hetken tilanne

**Julkaisurytmi:** 1 Feed-postaus / viikko + Storyt vapaasti + 1–2 Reels / viikko

**Seuraava julkaisu:** katso [SCHEDULE.md](SCHEDULE.md) — ⏳-merkityt ovat työn alla, 📅 suunniteltuja.

**Tilin tarkoitus:** Järvenpään Jousiampujien epävirallinen yhteisötili, joka tuo näkyväksi seuran arkea, kilpailuja ja lajia laajemmin. Täydentää seuran virallista tiliä [@jarvenpaan_jousiampujat](https://www.instagram.com/jarvenpaan_jousiampujat/), ei korvaa sitä.
